# 简介
这个项目是本人的毕业设计，目的是做一个类似Pinterest的图片分享应用。目前提供iOS版本客户端，后台由Django打造。
# 依赖关系
* 后台依赖：
    - Django Framework
    - Django-piston
    - Django-relationships
    - MySQL
    - libjpeg, libjpeg-dev
    - MySQL-Python
* 客户端依赖：
    - iOS 4.3以上
    - GPUImage
